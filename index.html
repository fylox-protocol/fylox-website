Exploring real-world payment infrastructure for Pi

Fylox is an early-stage protocol prototype enabling QR-based Pi payments. The current MVP focuses on testing core payment flows within the Pi ecosystem.

Protocol Vision

The Fylox protocol is designed with a modular architecture of five layers. Currently, we focus on the core payment layer. Future releases will expand into reputation, governance, integrations, and AI-powered utilities.

  • Layer 1: Core Payment (MVP)
  • Layer 2: Reputation (future)
  • Layer 3: Governance (future)
  • Layer 4: Integrations (future)
  • Layer 5: AI Utilities (future)
